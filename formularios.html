Ejercicio 1 Ir al formulario
Crea una página html con un formulario para obtener información personal sobre un usuario, los campos a incluir son los siguientes:
  1. Nombre
  2. Dirección
  3. NIF, comprobar que el documento introducido sea correcto.
  4. Fecha de nacimiento con formato dd/mm/aaaa.
  5. Código postal (numérico)
  6. País, será una lista con 6 paises.
  7. Género, será un radiobutton con tres opciones.
    Esta tercera opción se debe a que el usuario puede que no quiera decir su género por diversos motivos.
  8. Preferencias, tendrás 3 checkbox a elegir con tres colores.
  9. Teléfono de 9 dígitos y numérico.
  10. E-Mail
  11. Contraseña
  12. Verificación de la contraseña
Nota: Todos los campos son requeridos excepto el de las preferencias. Esto se debe a que se debe pulsar obligatoriamente el checkbox que se requiere aunque se hayan pulsado otros.
El formulario tendrá dos botones Enviar y Borrar, al pulsar Enviar la página mostrará la información que el usuario ha escrito. Para realizar el envío del formulario todos los campos deben de ser correctos. Los campos que no sean correctos aparecerán con el fondo rojo y no debe permitir el envío del formulario.
Ejercicio 2 Ir al formulario
Crear una página en la que se mostrarán una lista de elementos y dos botones:
  • Añadir: Al pulsarlo, se dará de alta un nuevo elemento en la lista y se solicitará el texto a introducir.
  • Eliminar: Al pulsarlo, se eliminará el último nodo. La lista siempre tendrá al menos un nodo en ella.
Nota: El texto del nuevo elemento se introducirá mediante un textarea, que estará dentro de un formulario. Para que se introduzca el elemento, el usuario deberá enviar el formulario. Dicho formulario solo será visible cuando se pulse el botón Añadir y se ocultará cuando se envíe el formulario.
Ejercicio 3 Ir al formulario
Realiza una página web que contenga dos o más párrafos, contendrá también:
  • Input de tipo text para introducir el id del párrafo a modificar
  • 3 botones:
    • Agrandar: Ampliará el tamaño del párrafo seleccionado en 0.05em hasta un máximo de 2em.
    • Reducir: Reducirá el tamaño del párrafo seleccionado en 0.05em hasta un mínimo de 0.9em.
    • Original: Restablecerá el tamaño al tamaño original.
Apunte: El tamaño de la fuente predeterminado para los párrafos es 1em. Sin embargo, si se quiere sacar ese valor mediante JavaScript, aparecerá vacío.
